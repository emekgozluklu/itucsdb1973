Parts Implemented by Şahin Akkaya¶

In this page, you will find information about using the application. The sections for this page are:

  • Sign up

  • Login

  • Search for a movie

  • Discover movies

  • View movie details

  • Add data

1. Sign up¶

In order to sign up to the site you need to go the sign up page. After filling the form with required information you can press “Sign up” button to continue.

2. Login¶

After signing up, you can visit the login page to log in to the site.

3. Search for a movie¶

You can press the “Search movies” button in the navigation bar to search movies. In this page you can search movies by name. You can also filter the results by genre.

4. Discover movies¶

In “Discover” page, you can view movies with their titles, release dates, languages, and a short description about them. You can click the title of a movie if you want to see more details.

5. View movie details¶

When you click on the title of a movie, you will be directed to page which shows more details about the movie. If you are logged in, you can add comment in this page. If you are just a guess, you can only view the details of the movie and comments made to the movie. Only admins are authorized to modify the details of movie or deleting, pinning-unpinning comments.

6. Add data¶

When you hover over “Add data” section at the navigation bar you will see 2 options pops up:

  • Add movie

  • Add genre

..warning You can not view or visit these pages if you are not an admin.

6.1 Add movie¶

In this page you can fill in the form to add a new movie to the application.

6.2 Add genre¶

You can also add new genres to the application which are shown as multiple selection list at “Add movie” page.
